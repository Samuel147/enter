## ✨ 核心功能

### 学生端（4个页面）
1. **仪表盘** - 入学进度、时间线、快速操作
2. **我的申请** - 申请列表、状态筛选
3. **新建申请** - 表单提交、附件上传
4. **个人信息** - 学生档案展示

### 教师端（4个页面）
1. **审批中心** - 待审批、统计卡片、快速审批
2. **审批历史** - 审批记录、处理时间
3. **学生管理** - 学生列表、搜索功能
4. **数据统计** - 统计图表、效率分析

### API 接口（3个）
1. `POST /api/auth/login` - 用户登录
2. `GET/POST /api/applications` - 申请管理
3. `GET/PUT /api/approvals` - 审批管理
