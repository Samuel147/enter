# 智慧新生入学系统

一个基于 Next.js 和 Tailwind CSS 的现代化新生入学管理系统。

## 功能特性

### 学生端
- 📊 入学流程追踪：实时查看入学进度，时间线展示每个环节状态
- 📝 申请管理：提交、查看各类入学相关申请（宿舍、课程、缓交学费等）
- 👤 个人信息：查看和管理个人档案信息
- 🔔 实时通知：及时了解申请状态变化

### 教师端
- ✅ 审批中心：快速处理学生申请，支持批量操作
- 📈 数据统计：查看审批效率、申请类型分布等统计数据
- 👥 学生管理：查看学生信息和入学进度
- 📋 审批历史：追溯所有审批记录

## 技术栈

- **前端框架**: Next.js 14 (App Router)
- **UI 框架**: Tailwind CSS
- **语言**: TypeScript
- **状态管理**: React Hooks + Context
- **API**: Next.js API Routes
- **图片资源**: Unsplash

## 项目结构

```
enrollment-system/
├── src/
│   ├── app/                    # Next.js App Router
│   │   ├── (auth)/            # 认证相关页面
│   │   ├── (student)/         # 学生端页面
│   │   ├── (teacher)/         # 教师端页面
│   │   └── api/               # API 路由
│   ├── components/            # React 组件
│   │   ├── auth/              # 认证组件
│   │   ├── student/           # 学生端组件
│   │   ├── teacher/           # 教师端组件
│   │   ├── common/            # 通用组件
│   │   └── ui/                # UI 基础组件
│   ├── lib/                   # 工具函数
│   ├── types/                 # TypeScript 类型定义
│   └── data/                  # 模拟数据
```

## 快速开始

### 安装依赖

```bash
npm install
# 或
pnpm install
```

### 启动开发服务器

```bash
npm run dev
# 或
pnpm dev
```

访问 http://localhost:3000 查看应用。

### 构建生产版本

```bash
npm run build
npm start
```

## 开发说明

### 添加新页面

1. 在 `src/app/(student)` 或 `src/app/(teacher)` 下创建新文件夹
2. 创建 `page.tsx` 文件
3. 在侧边栏添加导航链接

### 添加新 API

1. 在 `src/app/api` 下创建对应的路由
2. 实现 GET/POST/PUT/DELETE 方法
3. 在前端组件中调用 API

### 添加新组件

1. 在 `src/components` 对应目录下创建组件文件
2. 使用 TypeScript 定义 Props 类型
3. 遵循 Tailwind CSS 样式规范

## 样式指南

- 使用 Tailwind CSS 类名
- 遵循设计系统的颜色规范（primary 色系）
- 保持统一的间距和圆角
- 使用 flex 和 grid 进行布局
- 响应式设计（mobile-first）
